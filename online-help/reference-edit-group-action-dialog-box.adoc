= 編輯群組動作對話方塊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用"編輯群組動作"對話方塊來編輯您為群組建立的群組動作。

您必須OnCommand 具備「管理員」或「儲存管理員」角色。

* *行動名稱*
+
顯示群組動作的名稱。

* *說明*
+
顯示群組動作的說明。

* *群組*
+
顯示所選群組的名稱。

* *行動類型*
+
顯示群組動作的類型。您無法變更動作類型。不過、您可以修改用來設定群組動作的參數。





== 命令按鈕

* *儲存*
+
儲存您對群組動作所做的變更。

* *取消*
+
關閉「編輯群組動作」對話方塊、而不儲存變更。
