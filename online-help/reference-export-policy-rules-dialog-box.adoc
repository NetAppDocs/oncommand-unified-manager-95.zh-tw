= 匯出原則規則對話方塊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「匯出原則規則」對話方塊會顯示有關匯出原則、驗證傳輸協定、以及在屬於儲存虛擬機器（SVM）的磁碟區上啟用存取的詳細資料。您可以使用篩選器自訂匯出原則規則清單中的資訊顯示。根據預設、資訊會根據索引欄進行排序。

* *索引*
+
顯示指派給匯出原則規則的索引。這是唯一的數字。

* *存取傳輸協定*
+
顯示為匯出原則規則啟用的傳輸協定。

* *用戶端配對*
+
顯示具有存取屬於SVM之磁碟區資料權限的用戶端。

* *唯讀存取*
+
顯示用於讀取SVM所屬磁碟區資料的驗證傳輸協定。

* *讀寫存取*
+
顯示驗證傳輸協定、用於在屬於SVM的磁碟區上讀取或寫入資料。
