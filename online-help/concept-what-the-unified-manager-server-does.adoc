= Unified Manager伺服器的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager伺服器基礎架構由資料收集單元、資料庫和應用程式伺服器組成。它提供基礎架構服務、例如探索、監控、角色型存取控制（RBAC）、稽核和記錄。

Unified Manager會收集叢集資訊、將資料儲存在資料庫中、並分析資料、以查看是否有任何叢集問題。
