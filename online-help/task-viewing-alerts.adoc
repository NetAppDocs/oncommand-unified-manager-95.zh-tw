= 檢視警示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從「組態/警示」頁面檢視針對各種事件所建立的警示清單。您也可以檢視警示內容、例如警示說明、通知方法和頻率、觸發警示的事件、警示的電子郵件收件者、以及受影響的資源、例如叢集、集合體和磁碟區。



== 開始之前

您必須OnCommand 具備「操作員」、「資訊管理員」或「儲存管理員」角色。



== 步驟

. 在左側導覽窗格中、按一下*組態*>*警示*。
+
警示清單會顯示在「組態/警示」頁面中。
