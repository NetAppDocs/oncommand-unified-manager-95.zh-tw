= 診斷使用者功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
診斷存取的目的是讓技術支援人員協助您進行疑難排解、您只能在技術支援人員的指示下使用。

診斷使用者可在技術支援指示下執行OS層級命令、以進行疑難排解。
